Doctor Appointment Booking System Ongoing

An advanced AI-powered solution designed to streamline document processing and intelligent data retrieval. Supports multiple file formats, uses preprocessing for OCR, and features a custom CNN model for document classification. Key information is stored for semantic and keyword-based search, with an interactive chatbot for conversational access.

Key Features:
  • Role-based Authentication – Implemented three levels of secure login (Patients, Doctors, Admins) with JWT authentication.
  • Patient Portal – Patients can register, log in, book appointments, and manage existing bookings.
  • Doctor Dashboard – Doctors can view appointments, track earnings, and update their profile through a dedicated dashboard.
  • Admin Panel – Admins can manage appointments, oversee doctor profiles, and maintain platform integrity.
  • Online Payment Integration – Integrated secure payment gateway for patients to pay consultation fees online.
  • Responsive UI – Designed with React and Tailwind CSS for a seamless cross-device experience.
